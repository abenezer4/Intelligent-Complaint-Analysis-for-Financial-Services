# RAG Complaint Chatbot

This project implements a Retrieval-Augmented Generation (RAG) chatbot to analyze customer complaints for CrediTrust Financial.

## Overview
- Analyzes customer complaints across multiple financial products
- Uses semantic search to retrieve relevant complaint narratives
- Generates insights using a language model
- Provides a user-friendly interface for non-technical stakeholders

## Setup
```bash
pip install -r requirements.txt
```

## Usage
```bash
python app.py
```
